## 🗄️ Banco de Dados

### Tabelas Criadas:

#### `choppinho.users`
```sql
- id (UUID, PK)
- phone_number (VARCHAR, UNIQUE)
- first_name (VARCHAR)
- email (VARCHAR)
- nicknames (TEXT[], até 10)
- personality_mode (VARCHAR: default/offensive/light_zen)
- notification_preferences (JSONB)
- created_at, updated_at
```

#### `choppinho.auth_tokens`
```sql
- id (UUID, PK)
- phone_number (VARCHAR)
- token (UUID, UNIQUE)
- pin_code (VARCHAR, 6 dígitos)
- used (BOOLEAN)
- expires_at (TIMESTAMPTZ, 15 min)
- created_at
```

#### `choppinho.strava_tokens`
```sql
- id (UUID, PK)
- user_id (UUID, FK → users)
- athlete_id (BIGINT)
- access_token (TEXT, encrypted)
- refresh_token (TEXT, encrypted)
- expires_at (TIMESTAMPTZ)
- last_sync_at
- created_at, updated_at
```

#### `choppinho.activities`
```sql
- id (UUID, PK)
- user_id (UUID, FK → users)
- strava_activity_id (BIGINT, UNIQUE)
- name (VARCHAR)
- distance_meters (DECIMAL)
- moving_time_seconds (INTEGER)
- average_speed (DECIMAL)
- start_date (TIMESTAMPTZ)
- created_at, updated_at
```

#### `choppinho.race_registrations`
```sql
- id (UUID, PK)
- user_id (UUID, FK → users)
- race_name (VARCHAR)
- race_date (DATE)
- distance_km (DECIMAL)
- race_type (VARCHAR: run/triathlon/ironman)
- location (VARCHAR, opcional)
- bib_number (VARCHAR, opcional)
- target_time (VARCHAR, HH:MM:SS)
- notes (TEXT)
- finish_time (VARCHAR, resultado)
- position_overall (INTEGER)
- position_category (INTEGER)
- created_at, updated_at
```
